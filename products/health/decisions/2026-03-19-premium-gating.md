# Decision: Launch Fully Unlocked

**Date:** 2026-03-19
**Product:** Keel (Health)
**Status:** Decided

## Decision
Launch with all features fully unlocked. Define premium tiers post-launch based on early user feedback.

## Rationale
- No premium gating strategy defined yet (#12)
- Defining tiers before having users would be guesswork
- Launching free removes a blocker from the critical path
- Real usage data will inform what's worth gating
- Can add premium gating in a post-launch update once patterns emerge
